Sobre Este Projeto

Como Começou

Este projeto nasceu em 2024, inspirado pelo lançamento do filme Minecraft. Começou como um simples teste para explorar como o jogo poderia ser usado em atividades educativas.

Minha Jornada

Desenvolvi este projeto sozinha, desde a concepção inicial até a implementação. Foi uma experiência de aprendizado onde combinei minha paixão por Minecraft com interesse em métodos de ensino inovadores.

O Experimento

A ideia inicial era testar se:

  • O jogo realmente poderia ensinar conceitos práticos
  • Seriam possíveis atividades estruturadas dentro do sandbox
  • Os resultados justificariam o tempo investido

Descobertas

Através deste projeto pessoal, descobri que:

  • Minecraft é excelente para ensinar geometria básica
  • A construção colaborativa desenvolve habilidades sociais
  • O sistema de crafting ensina sequência lógica
  • Os jogadores aprendem sem perceber que estão estudando

Próximas Etapas

Mesmo sendo um projeto individual, pretendo:

  • Documentar melhor minhas observações
  • Criar tutoriais para outros interessados
  • Testar com pequenos grupos
  • Compartilhar meus resultados na comunidade
